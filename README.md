## 使用

- 初始化项目

```Basic
npm create mine
```

## 当前支持的预设

### 项目模板预设

- `vue`
- `react`
- `uni-app`
- `electron`
- `node`

### 安装工具预设

- `npm`
- `cnpm`
- `pnpm`
- `yarn`

## 项目解析

- [项目模板地址](https://github.com/biaov/project-template)：不同分支表示不同模板

### 技术栈

- `Vite` + `TypeScript` + `Node`

### 依赖特性

- `chalk`: 字体颜色
- `commander`: 命令
- `download-git-repo`: 下载模板工具
- `inquirer`: 用户交互
- `log-symbols`: 图标
- `ora`: 动画效果
- `chokidar`: 监听文件更改
- `vite`: 项目构建工具
- `typescript`: 编程语言
- `prettier`: 格式化工具
